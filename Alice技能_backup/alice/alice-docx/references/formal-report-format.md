# 正式報告格式實測數據

取自 `115年度萬山、寶山、來義等五處大規模崩塌地區監測計畫_工作執行計畫書_期中參考版.docx` Table 1 (6×8 基本資料表) 的實際格式分析。

## 量測結果 (2026-06-11)

### 字型
- 表頭 run font.name: 標楷體
- 資料列 run font.name: None（繼承 標楷體）
- 東亞字型 (w:eastAsia): None（預設繼承）
- font.size: None（繼承段落樣式，約 12pt）
- font.bold: None（無粗體）

### 表頭底色
- w:shd fill: F2F2F2
- w:shd color: auto

### 邊框 (w:tcBorders)
- top/left/bottom/right: sz=4, color=000000, val=single

### 對齊
- 水平 (w:jc): None（預設靠左）
- 垂直 (w:vAlign): center

### 表格樣式
- table.style.name: Normal Table

### 頁面設定
- 方向: portrait（直式）
- 紙張: 推測 A4

## 其他觀察
- 無交替行底色
- 無合併儲存格（簡單表）
- 資料儲存格無底色（白色）
- 表頭文字為黑色
